# Gerenciador de Professores

Repositório dedicado ao desenvolvimento da aplicação de gerenciamento de professores, parte do projeto final da disciplina de Desenvolvimento de Sistemas Web do curso TADS. O projeto utiliza uma arquitetura baseada em **Spring Boot (Webservice REST)** e **Banco de Dados H2**, com o frontend consumindo a API via **Fetch API**.

## 🛠 Tecnologias Utilizadas
- **Backend:** Java 17+, Spring Boot 4.0.x (Spring Web, Spring Data JPA, Spring Validation).
- **Banco de Dados:** H2 Database (em memória).
- **Frontend:** HTML5, CSS3, JavaScript (Fetch API).
- **Versionamento:** Git & GitHub.

## 📋 Funcionalidades
- [x] **CRUD Completo:** Listagem, inclusão, alteração e exclusão de professores.
- [x] **Validação de Dados:** Validações rigorosas no backend (@NotBlank, @Size).
- [x] **Interface Dinâmica:** Consumo de API RESTful via JavaScript.
- [x] **Persistência:** Integração com banco de dados em memória para testes funcionais.

## 🚀 Como Executar
1. Clone este repositório:
   `git clone https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git`
2. Certifique-se de ter o JDK instalado.
3. Importe como um projeto Maven na sua IDE (IntelliJ ou Eclipse).
4. Execute a classe `ProfessoresApplication`.
5. Acesse no navegador: `http://localhost:8080/`
